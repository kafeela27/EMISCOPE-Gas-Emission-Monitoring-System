# EMISCOPE – Real-Time Gas Emission Monitoring System

## Project Overview

EMISCOPE is an IoT-based gas emission monitoring system developed to monitor Carbon Monoxide (CO) emissions in real time using an ESP32 microcontroller and MQ-7 gas sensor. The system provides live data visualization through a web dashboard with secure user authentication.

## Technologies Used

- React.js
- HTML5

- Firebase
- ESP32
- MQ-7 Gas Sensor
